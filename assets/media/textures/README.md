# Texture provenance

## Leaderboard junkyard dirt

`leaderboard-junkyard-dirt-{albedo,normal,arm}.webp` are web-optimized 2K derivatives of Poly Haven's
[Dirt](https://polyhaven.com/a/dirt) material by Charlotte Baglioni. The source asset is dedicated to the
public domain under [CC0](https://polyhaven.com/license).

- Albedo source: `dirt_diff_2k.jpg`
- Normal source: `dirt_nor_gl_2k.jpg` (OpenGL orientation)
- AO/roughness/metal source: `dirt_arm_2k.jpg`

The source downloads were retrieved from `dl.polyhaven.org` through the official Poly Haven API and verified
against the MD5 checksums published by that API before conversion to WebP.
